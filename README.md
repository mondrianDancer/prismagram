- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See User profile
- [ ] Follow / Unfollow User
- [ ] Edit my profile
- [ ] Create account
- [ ] Login
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
