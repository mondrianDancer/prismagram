# prismagram

Instagram Clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See User profile
- [ ] Follow / Unfollow User
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
